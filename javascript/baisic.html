자바스크립트 기본 규칙

  1. 하나의 실행문이 끝나면 마지막에 세미콜론(;)을 붙여줍니다.
    세미콜론(;)을 붙여 주지 않아도 상관은 없습니다.

    var num = 5;
    document.write(num);

  2. 대소문자를 구분합니다.

    var num = 5;
    document.write(num); (O)
    document.write(Num); (X)

  3. 가독성을 위해 들여쓰기를 사용합니다.

    4. var avgDistance = function() {
    var speed - distance * time;
    }

  4. 주석(comment) 처리 방법에는 한글 주석 처리와 여러 줄 주석처리가 있습니다.

    5. //한 줄 주석처리 방법입니다.

    주석 처리 방법입니다.*/
